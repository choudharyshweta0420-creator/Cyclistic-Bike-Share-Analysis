# 🚴 Cyclistic Bike Share Analysis

## Project Overview

This project analyzes Cyclistic bike-share data to understand how annual members and casual riders use bikes differently.

The purpose of this analysis is to identify rider behavior patterns and provide recommendations that can help Cyclistic increase annual memberships.

---

## Business Question

**How do annual members and casual riders use Cyclistic bikes differently?**

The analysis focuses on understanding differences in:
- Ride frequency
- Ride duration
- Weekly riding patterns
- Monthly trends
- Bike preferences
- Popular starting stations

---

## Tools Used

- Python
- Pandas
- Matplotlib
- Kaggle Notebook
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Data Visualization

---

## Data Cleaning Process

The dataset was prepared for analysis through the following steps:

- Checked dataset structure and size
- Reviewed column names and data types
- Checked missing values
- Removed duplicate records
- Converted date and time columns into the correct format
- Calculated ride duration
- Removed invalid trips

---

## Analysis Performed

The following analyses were completed:

### 1. Rider Type Analysis
Compared the number of rides taken by:
- Annual members
- Casual riders

### 2. Ride Duration Analysis
Calculated and compared average ride length between rider groups.

### 3. Weekly Usage Analysis
Analyzed ride patterns by day of the week.

### 4. Monthly Trend Analysis
Examined seasonal changes in bike usage.

### 5. Bike Type Analysis
Compared bike preferences between rider groups.

### 6. Station Analysis
Identified the most frequently used starting stations.

---

## Key Findings

The analysis showed that:

- Annual members demonstrate more consistent riding behavior.
- Casual riders tend to use bikes more recreationally.
- Riding patterns change depending on the day of the week and season.
- Certain bike types and stations have higher demand.

---

## Business Recommendations

Based on the findings:

- Create targeted membership promotions for casual riders.
- Offer trial memberships to encourage conversion.
- Promote membership benefits such as convenience and cost savings.
- Use rider behavior data to create personalized marketing campaigns.
- Focus marketing efforts during high casual-rider usage periods.
